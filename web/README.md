Mobile-first React app (Vite + TypeScript)

Quick start (Windows PowerShell):

```powershell
cd C:\Users\elias\IdeaProjects\demo1\web
npm install
npm run dev
```

Open the URL printed by Vite (usually http://localhost:5173). To build for production:

```powershell
npm run build
npm run preview
```

Notes:
- This is a minimal scaffold focused on mobile-first CSS and touch-friendly UI.
- If you prefer JavaScript instead of TypeScript, recreate using `npm create vite@latest web -- --template react`.
- Consider adding Tailwind CSS or other UI libraries if you need rapid design systems.
